# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 1.0.x   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability in Sinedrio Danai, please report it
responsibly by opening a **private security advisory** on this repository.

- Do **not** open a public issue for security vulnerabilities.
- We aim to acknowledge reports within 48 hours and provide a fix within 7 days.

### API Keys

Never commit API keys to source control. Use the in-app *API Settings* dialog
or environment variables to supply keys at runtime.
